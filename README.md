# 🏨 Hotel Booking React App

A modern and responsive **hotel booking web application** built using React. This project allows users to explore rooms, view details, and manage bookings with a clean UI.

---

## 🚀 Features

* 🏠 Home page with featured rooms
* 🛏️ Room details page
* 📅 Booking functionality
* 📖 My Bookings section
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* CSS / Tailwind
* React Router

---

## 📂 Project Structure

```
src/
│── components/
│── pages/

```

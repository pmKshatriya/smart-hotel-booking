# 🏨 Smart Hotel Booking React App

A modern and responsive **hotel booking web application** built using React. This project allows users to explore rooms, view details, and manage bookings with a clean UI.

---

## ✨ Features

### 👤 User Side
- Browse all available hotel rooms
- Filter rooms by type, price range
- Sort rooms by price
- View detailed room information with image gallery
- Check-In / Check-Out booking form
- View and manage personal bookings
- Payment status tracking (Paid / Unpaid)
- Clerk Authentication (Login / Signup)

### 🏢 Hotel Owner Side
- Secure owner dashboard
- Add new rooms with images, type, price & amenities
- List and manage all rooms
- Toggle room availability
- Track recent bookings with payment status
- View total bookings and total revenue
